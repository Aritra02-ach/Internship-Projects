Sorting Visualizer

In this project,the unsorted numbers are taken as input and stored in the form of an array. The unsorted array undergoes bubble sort algorithm and presented in the form of graphs.

The unsorted array is reprsented using Bargraph and Linegraph and after sorting the array is again represented through the graphs. This graphical interface helps the user to identify the difference betwwen the unsorted and sorted array.
